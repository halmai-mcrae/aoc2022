# 🎄 Advent of Code 2022 🎄

View my solutions to the [Advent of Code](https://adventofcode.com/) challenge.

## Installation

Clone this repository.
```
git clone https://github.com/halmai-mcrae/aoc2022/edit/main/README.md
```
Then:
```
cd aoc2022
```
Run:
```
npm i
```

## Running in dev mode

```
npm start <day>
```

Example:

```
npm start 1
```
